The Lists

Lists are Annotation sources intended as gazetteers for lookup-based annotation of documents, i.e. annotating all words / phrases in a document that are contained in a particular list.

The Lists Menu

The Lists menu provides access to functions for managing and applying gazetteer Lists:
  • Create: Create a new List by providing entry data.
  • Load: Create a new List loading a text file from the local file system.
  • Load URL: Create a new List by loading a text file from an arbitrary URL.
  • Extract: Create a new List by extracting the values of a particular type of Annotations from the document currently selected.
  • Edit: Open the List editor dialog.
  • Apply: Select a gazetteer list and use it to annotate all words / phrases in a currently selected document that are contained in that List. This function is also available through the Apply List option in the Tools menu.

The Edit Lists Dialog

The dialog for List management is accessible through the Edit function in the Lists menu. On the right of the dialog, a list displays all existing Lists. In the middle, it displays the entries of the List currently selected in that list. Below the entries, three buttons provide functions for editing the List currently selected:
  • Union: Opens a dialog for selecting another List to be unioned with the one currently being edited. Above the Union and Cancel buttons, that dialog has a checkbox for specifying if the union operation should be case sensitive or not. The second List will not be changed by this operation.
  • Intersect: Opens a dialog for selecting another List to be intersected with the one currently being edited. Above the Union and Cancel buttons, that dialog has a checkbox for specifying if the intersect operation should be case sensitive or not. The second List will not be changed by this operation.
  • Subtract: Opens a dialog for selecting another List to be subtracted from the one currently being edited. Above the Union and Cancel buttons, that dialog has a checkbox for specifying if the subtraction operation should be case sensitive or not. The second List will not be changed by this operation.
At the top of the dialog, a line of buttons provides the functionality for managing the Lists:
  • Create: Create a new List.
  • Clone: Create a new List, copying the entries of the one currently selected.
  • Load: Create a new List loading a text file from the local file system.
  • Load URL: Create a new List by loading a text file from an arbitrary URL.
  • Extract: Create a new List by extracting the values of a particular type of Annotations from the document currently selected.
  • Delete: Delete the List currently selected.

Applying Lists

Whenever a List is to be applied, there are two checkboxes providing additional influence on the behavior of the List based annotation:
  • Case Sensitive: If checked, the List lookups are case sensitive, otherwise they are not.
  • Overlapping Matches: If checked, every word or phrase in the document is annotated if it is contained in the List, regardless of overlaps. Otherwise, only the longest match is annotated from each set of overlapping matches.
